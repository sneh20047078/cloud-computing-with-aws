# Cloud Computing with AWS

This repository contains hands-on AWS assignments and projects completed as part of a Cloud Computing with AWS training program. All infrastructure is deployed in the **Asia Pacific (Mumbai) — ap-south-1** region.

---

## Skills Demonstrated

- AWS IAM (Users, Groups, Roles, Policies)
- Amazon VPC (Subnets, Route Tables, Internet Gateway, Security Groups)
- Amazon EC2
- Amazon S3
- Static Website Hosting
- Amazon CloudFront
- AWS RDS (MySQL)
- AWS Lambda (Python)
- Amazon API Gateway
- Amazon CloudWatch
- Serverless Architecture
- AWS CLI
- Git & GitHub
- Cloud Security Fundamentals

---

## Repository Contents

### Week 1
- AWS Account Setup
- IAM Configuration (Users, Groups, Policies)
- Core AWS Services Overview
- S3 Static Website Hosting
- IAM & S3 Hands-on Mini Project

### Week 2
- EC2 Instance Deployment
- Application Hosting on EC2
- S3 Static Website Hosting
- CloudFront CDN Deployment
- Portfolio Website Hosting

### Week 3
- Custom VPC with Public & Private Subnets (CIDR: 10.0.0.0/16)
- Internet Gateway, Route Tables, and Security Group configuration
- EC2 Instances in isolated network tiers (Public + Private)
- MySQL 8.4.8 RDS Instance in private subnet (db.t3.micro)
- EC2-to-RDS connectivity verified via SQL queries
- Serverless Contact Form — Lambda (Python 3.12) + API Gateway HTTP API
- POST /contact endpoint tested via Postman (200 OK)
- Lambda execution monitored in CloudWatch (Billed Duration: 110ms, Memory: 36MB)
